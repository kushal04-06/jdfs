## Project Description

The **Student ERP System** is a simple web-based application that demonstrates the basic functionality of a college ERP portal. It allows students to register, log in, manage their profile, view academic details, and receive notifications.

This project is built using **HTML, CSS, and JavaScript** in a single file for easy demonstration.

---

## Features

### Student Registration

Students can create a new account by entering:

* Full Name
* Email
* Username
* Password

### Secure Login

Students log in using:

* Username or Email
* Password

### Forgot Password

A **Forgot Password** option is available for users who forget their password.

### Student Dashboard

After login, students are redirected to a **dashboard** where they can access different modules.

### Profile Management

Students can update personal details such as:

* Name
* Phone
* Address

### Academic Details

Students can check:

* Subjects
* Marks
* Attendance

### Notifications

Important announcements from the institution are displayed.

### Logout

Students can securely log out from the system.
